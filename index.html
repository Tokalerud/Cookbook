💾 Сохранить базу данных

Выгрузить все рецепты в файл JSON на ваше устройство. Файл можно использовать для резервной копии или переноса на другое устройство.

📂 Загрузить базу данных

Выберите рецепты и скачайте их одним файлом — HTML или PDF. Файл можно открыть без приложения.

📥 Загрузка рецептов

Загрузите HTML-файл с рецептами (полученный от другого пользователя или выгруженный ранее). Дубликаты добавлены не будут.
