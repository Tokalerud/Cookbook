## 📲 Установка на телефон

### Android (Chrome)
1. Откройте сайт в **Chrome**
2. Нажмите кнопку **📲** в правом верхнем углу
3. Подтвердите установку — приложение появится на рабочем столе

### iPhone / iPad (Safari)
1. Откройте сайт в **Safari**
2. Нажмите кнопку **Поделиться** (квадрат со стрелкой вверх) внизу экрана
3. Выберите **«На экран "Домой"»**
4. Нажмите **Добавить** — приложение появится на рабочем столе

### Внимание!!! Если открыли ссылку в Telegram, VK или другом приложении?
Встроенные браузеры не поддерживают установку. Нажмите кнопку **🌐** в правом верхнем углу приложения — оно откроется в полноценном браузере, откуда уже можно установить.

Или откройте ссылку вручную: скопируйте адрес и вставьте в Chrome (Android) или Safari (iPhone).

После установки работает **офлайн** — без интернета.

## ✨ Возможности

### 🍽 Рецепты
- Хранение рецептов с фото, ингредиентами, шагами и заметками
- Импорт рецептов с сайтов по ссылке (russianfood.com, povarenok.ru, edimdoma.ru, gastronom.ru и другие через Schema.org)
- Автоматический расчёт калорий и БЖУ по ингредиентам
- Пересчёт порций — меняйте количество, ингредиенты пересчитываются автоматически
- Избранное ❤️, категории, поиск
- Вид сеткой или списком
- Экспорт в HTML и PDF

### 📅 Планировщик меню
- Планирование питания по дням и неделям
- Учёт калорий и БЖУ за день с прогресс-барами
- Настройка приёмов пищи (завтрак, обед, ужин и другие)
- Калькулятор дневной нормы по формуле Миффлина-Сан Жеора
- Копирование дня на любую неделю
- Добавление блюд из рецептов, базы продуктов или вручную

### 🛒 Список покупок
- Формирование списка из рецепта одной кнопкой
- Умное объединение одинаковых продуктов
- Быстрый ввод голосом 🎤 или текстом
- Группировка по категориям
- Экспорт и шаринг

### 🥦 База продуктов
- Встроенная база ~640 продуктов с калориями и БЖУ
- Личная база — добавляйте свои продукты
- Автоматическое сохранение ингредиентов при создании рецептов

### 🍳 Режим готовки
- Полноэкранный пошаговый режим
- Таймер с уведомлением и вибрацией
- Экран не гаснет во время готовки (Wake Lock)

## 💾 Данные

- Хранятся в **IndexedDB** браузера — только на вашем устройстве
- Резервная копия — HTML-файл, можно скачать и восстановить в любой момент
- Копия содержит все рецепты, планировщик, список покупок и настройки

---

## 🛠 Технологии

- Чистый **HTML + CSS + JS** — без фреймворков и зависимостей
- **IndexedDB** — локальное хранилище
- **Service Worker** — офлайн-режим, network-first стратегия
- **Web Share API** — шаринг рецептов и покупок
- **Web Speech API** — голосовой ввод
- **Wake Lock API** — экран не гаснет при готовке
- **Canvas API** — сжатие фотографий
- Один файл `index.html` — весь код внутри
